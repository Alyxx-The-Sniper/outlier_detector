# Outlier Detector

A Python toolkit for detecting outliers using statistical and machine learning methods with visualizations.

## Features

- IQR and Z-Score detection
- Isolation Forest
- Local Outlier Factor (LOF)
- One-Class SVM
- DBSCAN
- Autoencoder-based detection
- PCA, t-SNE, UMAP visualizations
